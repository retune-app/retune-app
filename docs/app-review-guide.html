App Overview

Retune helps users reprogram their subconscious mind through personalized audio affirmations and guided breathing exercises.

"Breathe, Believe, Become"

Navigation

The app has two main tabs at the bottom:

Tab Icon Purpose
Breathe Wind icon (left) Guided breathing exercises
Believe Heart icon (right) Affirmation library
+ Button Gold circle (center) Create new affirmation

Breathe Tab

How to Use Breathing Exercises

  1. Select a technique - Tap the card at the top to choose:
    Box Breathing 4-7-8 Technique Coherent Breathing
  2. Set duration - Choose 1, 3, 5, or 10 minutes
  3. Optional audio - Select Off, Rain, or Voice
  4. Start - Tap the play button to begin
  5. Follow the guide - The circle expands and contracts with countdown timer

Controls During Session

  • Stop - End session early
  • Pause - Temporarily pause the exercise
  • Expand - Full-screen immersive mode

Believe Tab (Affirmation Library)

How to Use the Library

  1. Browse affirmations - Scroll through the card list
  2. Filter by category - Tap chips at top:
    All Favorites Mind Body Spirit Connection Achievement Home
  3. Play an affirmation - Tap the gold play button on any card
  4. Favorite an affirmation - Tap the heart icon on the card header
  5. Search - Use the search bar to find specific affirmations

Affirmation Cards

Each card displays:

  • Voice name - AI voice used (e.g., "Rachel", "Charlie")
  • Title - Affirmation name
  • Tags - Category labels (e.g., Career, Confidence)
  • Duration - Audio length
  • Play button - Gold circle to play audio

Creating an Affirmation (+ Button)

Step-by-Step Creation

  1. Choose mode - Auto (AI-generated) or Manual
  2. Select focus area - Mind, Body, Spirit, Connection, Achievement, or Home
  3. Add tags - Optional subcategories for organization
  4. Describe your goal - Enter what you want to achieve
  5. Select length - Short, Medium, or Long
  6. Generate Script - AI creates personalized affirmation text
  7. Create Affirmation - Generates audio and saves to library

Settings (Gear Icon)

Access via the gear icon on Breathe or Believe screens:

  • Preferred Name - Personalize app greetings
  • Analytics - View listening stats and streaks
  • Voice Settings - Choose preferred AI voice
  • Sound Library - Browse ambient sounds
